- Assembly -

Assembly language is a low-level programing language for a computer or other programmable device. Assembly language is converted into executable machine code by a utility program that's called an assmebler.

Each computer has a microprocessor that manges the computer's algorithems, logical and control activites. Each family of processors has its own set of instructions for handling variuos operaions. Processors understands only machine language instructions, however, machine language is too complex for using in software development. So, the assembly language is designed to make these instructions in symbolic code and more understandable form.
